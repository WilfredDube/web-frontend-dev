HTML Lists

There are 3 types of list in HTML: unordered lists, ordered lists and definition lists. The definition list arranges all items in the list as they appear in the dictionary.

HTML unordered list

To create an unordered list use the <ul> tag. Inside this tag add the <li> list tags. The ul tag has a type attribute which has 3 values: square, disc and circle.

Default type: disc
